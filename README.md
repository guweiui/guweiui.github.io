# 🎨 《[画室里的调色盘](https://guweiui.github.io/)》

老洋房的二楼被改成了画室，落地窗外爬满了常春藤，阳光透过枝叶的缝隙洒进来，在地板上投下斑驳的光影。林默的画架上总摆着一幅未完成的油画，画布上是一个穿着白裙子的女孩，站在一片向日葵花田里，笑容像阳光一样灿烂。

那个女孩是苏薇，楼下花店的老板。她每天都会送一束向日葵上来，插在画室的陶瓷花瓶里。林默会把画好的小速写送给她，有时是一只蜷缩在沙发上的猫，有时是窗外的常春藤，每幅画的角落都藏着一个小小的“薇”字。

他们是在一个雨天认识的，苏薇抱着一捆被淋湿的向日葵，站在画室门口敲门，问能不能借个地方避雨。林默给她倒了一杯热水，她坐在沙发上，用纸巾擦拭花瓣上的水珠，阳光透过云层洒进来，落在她的发梢上，像镀了一层金边。那天林默画了她的侧影，线条温柔，像一首没写完的诗。

上周苏薇的花店要装修，林默放下画笔帮她搬花盆。他不小心打碎了一个陶瓷花瓶，碎片溅在他的手背上，划出一道浅浅的伤口。苏薇赶紧找了创可贴给他贴上，指尖碰到他的伤口时，他的心跳漏了一拍，像被什么东西轻轻撞了一下。

昨天我去画室找林默，他正坐在画架前，画布上的向日葵花田里，女孩的身边多了一个男孩，他们手牵着手，笑容温暖。苏薇站在他身后，帮他调颜料，阳光透过玻璃窗洒进来，落在他们身上，像一幅流动的油画。

现在画室里的向日葵每天都会换一束新鲜的，画架上的油画已经完成了，挂在画室最显眼的位置，旁边写着“我的向日葵，我的光”。

金句： 爱情是一场温柔的创作，你是我调色盘里最鲜艳的颜色，让我的人生从此有了意义。
